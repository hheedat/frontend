## 本地开发

1. 使用本地服务器打开项目
```bash
# 使用 Python 的简单 HTTP 服务器
python -m http.server

# 或使用 Node.js 的 http-server
npx http-server
```

2. 在浏览器中访问 `http://localhost:8000` 查看项目
